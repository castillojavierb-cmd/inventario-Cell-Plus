# Sistema de Inventario

## 📌 Descripción
Este proyecto permite gestionar productos, ventas y usuarios con autenticación JWT.

## 🔐 Autenticación
- Login con token
- Rutas protegidas
- Uso de localStorage

## 📦 Funcionalidades
- Crear productos
- Editar productos
- Eliminar productos
- Vender productos

## ⚙️ Tecnologías
- React
- Node.js
- Express
- MySQL

## 🚀 Estado
Proyecto en desarrollo

## Scripts

cd frontend
npm run dev

cd backend
npm start
